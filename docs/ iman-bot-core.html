Impulsa tus publicaciones en TikTok con inteligencia y resonancia

¿Qué es Imán Bot Core?

Un sistema automatizado conectado a GitHub que activa procesos de visibilidad sobre publicaciones en TikTok, usando inteligencia de impulso modular.

📌 Cómo funciona

  1. Crea un issue en el repositorio con tu video
  2. El bot escucha y ejecuta el análisis
  3. Se activa el módulo de impulso (XOXO)
  4. Recibís feedback desde Discord y logs públicos
